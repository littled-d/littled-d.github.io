AJAX技术是前端常用的技术，JSON也是前后端交互最常用的数据格式；

AJAX

概念：AJAX(Asynchronous JavaScript And XML)：异步的JavaScript和XML

AJAX的作用：

  1. 与服务器进行数据交换：通过AJAX可以给服务器发送请求，并获取服务器响应的数据

  • 以前我们是，用户把请求发到一个Servlet，Servlet得到从该数据库访问出来的结果集合，Servlet把结果集合放进JSP够访问到的域里面，然后转发到指定JSP读取数据，进行一个动态展示；以前，是不得不用JSP，不用就得把Html代码用java的out操作一行一行的输出到前端页面；（人都给你输傻咯）

  • 现在呢，使用AJAX和服务器进行通信，就可以使用HTML+AJAX来替换JSP页面了；因为JSP也是后端写的，所以以前根本做不到前后端分离；有了AJAX就不一样了：前端用AJAX发出请求，后端用AJAX返回响应，最后这样一通信，整个项目也就完成了，就可以找前端小姐姐聚餐了(/嘿嘿);

  • 好处就是更加有利于前后端分离；方便交互，减少程序员工作量
  1. 异步交互：可以在不重新加载整个页面的情况下，与服务器交换数据并更新部分网页的技术，如：搜索联想、用户名是否可校验，等等…
    • 同步与异步：同步是线性，在请求的同时，用户不能操作别的，异步是开启一个新线程（类似，描述并不准确），用户可以做自己的事情，请求是后台悄悄进行；

AJAX的快速入门

1、编写AjaxServlet，并使用response输出字符串

@WebServlet("/ajaxServlet")
public class AJAXServlet extends HttpServlet {

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        System.out.println("欢迎来到ajax");
        resp.getWriter().println("welcome");
    }
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

2、创建XMLHttpRequest对象：用于和服务器交换数据

3、向服务器发送请求

4、获取服务器响应数据

// 此代码块放在html文件的<script></script> 标签内
//1、创建核心对象
var xhttp;
if (window.XMLHttpRequest) {
    xhttp = new XMLHttpRequest();
} else {
    // code for IE6, IE5
    xhttp = new ActiveXObject("Microsoft.XMLHTTP");
}
//2、发送请求  
// 此处路径写全路径，因为以后前后端分离，就是两个项目，会部署到不同的服务器上，所以不会写相对路径
// open(method, url, async) 
// method：请求的类型：GET 还是 POST；
// url：服务器（文件）位置；
// async：true（异步默认）或 false（同步）
xhttp.open("GET", "http://localhost/ajaxServlet", true);
xhttp.send();

//3、接收响应
xhttp.onreadystatechange = function() {
    if (this.readyState == 4 && this.status == 200) {
        alert(this.responseText);
    }
};

Axios异步框架

  • Axios是对于原生的AJAX进行封装，简化书写
  • Axios官网

Axios的快速入门

1、引入Axios的js文件

npm install axios 导包，将里面的axios.js放在项目文件中引入；

<script src="/axios.js"></script>

2、使用Axios发送请求，并获取响应结果

<script>
    axios({
        method:"get",   //get方法在请求头中拼装数据 ，post方法再加一个请求体
        url:"http://localhost/ajaxServlet"
    }).then(function (resp) {
        alert(resp.data+"123");
    });

    axios({
        method:"post",   //get方法在请求头中拼装数据 ，post方法再加一个请求体
        url:"http://localhost/ajaxServlet"
    }).then(function (resp) {
        alert(resp.data+"123");
    });
</script>

Axios请求方式别名

为了方便起见，Axios已经为所有支持的请求方式提供了别名。

例：get(url) 方法发起get请求 post(url,参数) 发起post请求

<script>
    axios.get("http://localhost/ajaxServlet").then(function (resp) {
        alert(resp.data+"123");
    });
    axios.post("http://localhost/ajaxServlet","key=value").then(function (resp) {
        alert(resp.data+"123");
    });
</script>

JSON

  • 概念：JavaScript Object Notation: JavaScript对象表示法

  • 由于其语法简单，层次结构鲜明，现多用于数据载体，在网络中进行数据传输

JSON的基础语法

1、定义：

var json = {"name":"zhangsan",
			"age":35,
            "work":"software enginner"
}
  • 数字直接写；字符串放在“引号”中；布尔值直接写；数组[方括号]，以逗号隔开；对象{花括号}，以逗号隔开；

2、获取值：

变量名.key；

JSON数据与Java对象的转换

  • 请求数据：JSON字符串转为Java对象；
  • 响应数据：Java对象转为JSON字符串；

使用的API是Fastjson，这是阿里提供的一个Java语言编写的高性能功能完善的JSON库，是目前Java语言中最快的JSON库，可以实现Java对象和JSON字符串的相互转换。

1、引入坐标：

<!-- https://mvnrepository.com/artifact/com.alibaba/fastjson -->
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>fastjson</artifactId>
    <version>1.2.79</version>
</dependency>

2、Java对象转为JSON

3、JSON字符串转为Java对象

public class JSONDemo {
    public static void main(String[] args) {
        Student student = new Student("s004", "老谢", 12, "计科");

        // java对象转为JSON对象
        String s = JSON.toJSONString(student);
        System.out.println(s);
        //结果：{"age":12,"college":"计科","name":"老谢","sNO":"s004"}
        
        //JSON对象转为Java对象
        Student student1 = JSON.parseObject(s, Student.class);
        System.out.println(student1);
        //结果：Student{SNO='s004', Name='老谢', Age=12, College='计科'}
    }
}
